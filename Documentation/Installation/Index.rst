Installation
============

Requirements
------------

- TYPO3 CMS 11.5
- Composer-based installation

Install extension
-----------------

Add the package to your project:

.. code-block:: bash

   composer require itx/csv-editor:^1.0

After installation, clear TYPO3 caches.

.. code-block:: bash

   ./vendor/bin/typo3 cache:flush
