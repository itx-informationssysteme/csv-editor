User Manual
===========

Open the CSV editor
-------------------

1. Go to **Filelist** in TYPO3 backend.
2. Navigate to the folder containing your CSV file.
3. Click the edit action for the CSV file.

Edit and save
-------------

- Use **Add row** / **Add column** to extend the table.
- Use **Remove row** to remove unwanted entries.
- Enable **Keep UTF-8 BOM** if needed.
- Save via the docheader **Save** button.

Notes
-----

- The editor writes semicolon-separated CSV data.
- Empty data rows are ignored when saving.
