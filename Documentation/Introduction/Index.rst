Introduction
============

The extension provides a custom CSV editor for TYPO3 backend users.

Features
--------

- Adds a dedicated edit action for CSV files in TYPO3 Filelist.
- Opens a table-like editor instead of the plain text editor.
- Supports adding/removing rows and columns.
- Preserves optional UTF-8 BOM on save.

Scope
-----

This extension targets TYPO3 CMS 11.5 projects and integrates with backend file handling.
